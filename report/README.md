# Report Archive

## Scopo

Questa cartella non e documentazione canonica per l'agent.
Contiene solo materiale storico:

- audit
- diagnostiche
- report di fix
- proposte tecniche
- note una tantum

## Regola di utilizzo

- `DNA/` e `README_OPERATIVO.md` restano la fonte operativa corrente
- `report/` serve solo come archivio
- i file qui dentro non devono essere trattati come fonte di verita se divergono dal codice o da `DNA/`

## Struttura

- `report/README.md`
  guida minima
- `report/archive/`
  documenti storici non canonici
